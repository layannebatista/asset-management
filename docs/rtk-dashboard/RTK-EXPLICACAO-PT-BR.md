# RTK explicado de forma simples

RTK é uma camada de apoio para usar melhor tokens em fluxos com IA. Em vez de mandar contexto demais, a ideia é organizar, reduzir e medir o que realmente ajuda.

## Por que isso importa

Tokens custam dinheiro e também influenciam velocidade. Se o contexto é grande demais, a análise fica mais cara e pode ficar menos objetiva.

## O que o dashboard mostra

| Métrica | Tradução |
|---|---|
| Tokens economizados | Quanto contexto foi evitado |
| Redução | Percentual de economia |
| Eficiência do modelo | Relação entre custo e resultado |
| ROI | Retorno estimado da otimização |
| Histórico | Evolução ao longo do tempo |

## Exemplo simples

Sem otimização, uma análise poderia usar muitos arquivos e tokens. Com RTK, o fluxo tenta enviar apenas o contexto mais relevante. O dashboard compara esses cenários e mostra a economia estimada.

## Para gestores

Use para entender tendência de custo e eficiência.

## Para devs

Use para identificar fluxos que mandam contexto demais e podem ser melhorados.

## Para apresentação

Confirme se os dados são reais, qual período foi usado e se há volume suficiente para conclusão.

## Limitação

ROI e economia são indicadores. Eles ajudam na decisão, mas não devem ser tratados como fechamento financeiro sem validação.
